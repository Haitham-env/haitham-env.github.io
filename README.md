# Haitham Al-Shahri Portfolio

نسخة مستقلة بالكامل من الموقع.

## التشغيل

افتح ملف `index.html` مباشرة في المتصفح.

ولتشغيله محليًا عبر خادم بسيط:

```bash
python -m http.server 8000
```

ثم افتح:

```text
http://localhost:8000
```

## النشر

ارفع ملف `index.html` إلى أي استضافة مواقع ثابتة، مثل:

- Cloudflare Pages
- Vercel
- Netlify
- GitHub Pages
- أي استضافة تدعم HTML

## التعديل

كل التصميم والنصوص والروابط موجودة داخل `index.html`:

- النصوص داخل قسم `<body>`.
- الألوان والتصميم داخل قسم `<style>`.
- رابط مشروع Power BI داخل بطاقة المشروع.
- البريد ورقم الهاتف ورابط LinkedIn في قسم التواصل.

لا يحتاج الموقع إلى قاعدة بيانات أو خادم خلفي.
